# cra-template-junior

<hr/>

This is a React template build for Junior Dev from a Junior.

Based on template: React Official template.

## Install & Start

<hr/>

Create React App with the template

```bash
yarn create react-app --template junior my-app
```

Start app, if you want

```bash
cd my-app
yarn start
```
